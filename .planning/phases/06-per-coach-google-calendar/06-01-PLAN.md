   - Export both functions as named exports

2. Create `src/lib/google/oauth.ts` with OAuth2 client factory:
   - Import `google` from `googleapis`
   - `SCOPES` constant: `["https://www.googleapis.com/auth/calendar"]`
   - `createOAuth2Client()`: returns `new google.auth.OAuth2(GOOGLE_CLIENT_ID, GOOGLE_CLIENT_SECRET, GOOGLE_REDIRECT_URI)` from env vars
   - `generateAuthUrl(coachId: string): string`: creates OAuth2Client, calls `client.generateAuthUrl({ access_type: "offline", scope: SCOPES, prompt: "consent", state: coachId })` -- CRITICAL: `prompt: "consent"` forces refresh_token on every auth (see Research Pitfall 1)
   - Export both functions as named exports

Follow the exact code patterns from 06-RESEARCH.md. Use double quotes per project convention (Biome).
