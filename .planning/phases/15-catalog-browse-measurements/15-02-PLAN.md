5. `passesFitsMeFilter(dress, student): boolean` per research L486-512:
   - slackLo = alterableSmaller ? 2 : 0
   - slackHi = alterableLarger ? 2 : 0
   - For each of chest/waist/hips: if caller dim is null OR dress min/max is null → continue (MATCH; no penalty)
   - Else: return false if `s < dMin - slackLo` OR `s > dMax + slackHi`
   - Length check: if dress.lengthCm and student.heightCm both set, abs(dress.lengthCm - expectedDressLengthForHeight(heightCm)) must be ≤ 8cm
6. `scoreDress(dress, student): number` per research L518-543:
   - Same slackLo/slackHi extraction
   - For each of chest/waist/hips: if caller dim null OR either dress bound null → nullDimensions += 1, continue
   - Else: compute center = (minWithSlack + maxWithSlack)/2, halfRange = (maxWithSlack - minWithSlack)/2 + 1, contribution = 1 - min(1, abs(s - center)/halfRange); score += contribution
   - Return `score - nullDimensions * 0.1`
   - Note: this CAN return a negative number for a dress with all three dressbounds null (-0.3). That is correct — penalize sparse listings.
7. `scoreToPercent(score: number): number` — `Math.min(100, Math.max(0, Math.round((Math.max(0, score) / 3) * 100)))`
8. `expectedDressLengthForHeight(heightCm: number): number` — `Math.round(heightCm * 0.45)`

```ts
// Build the StudentFitFields shape if caller has measurements
const callerFit: StudentFitFields | null = callerMeasurements
  ? {
      chestCm: callerMeasurements.chestCm,
      waistCm: callerMeasurements.waistCm,
      hipsCm: callerMeasurements.hipsCm,
      heightCm: callerMeasurements.heightCm,
    }
  : null;

// fitsMe filter — apply BEFORE sort + pagination
let filtered = dresses;
if (input.fitsMe && callerFit) {
  filtered = filtered.filter((d) => passesFitsMeFilter(d, callerFit));
}

// Sort — bestFit requires callerFit (validated earlier with BAD_REQUEST)
let sorted: typeof filtered;
if (input.sort === "bestFit" && callerFit) {
  sorted = [...filtered]
    .map((d) => ({ d, score: scoreDress(d, callerFit) }))
    .sort((a, b) => b.score - a.score)
    .map(({ d }) => d);
} else if (input.sort === "priceAsc") {
  sorted = [...filtered].sort((a, b) => a.competitionPrice - b.competitionPrice);
} else if (input.sort === "priceDesc") {
  sorted = [...filtered].sort((a, b) => b.competitionPrice - a.competitionPrice);
} else {
  // "newest" — already in createdAt desc order from findMany
  sorted = filtered;
}

const total = sorted.length;
const start = (input.page - 1) * input.limit;
const items = sorted.slice(start, start + input.limit);

// Attach per-card fit info ONLY when caller has measurements — used by BestFitBadge (15-05)
const itemsWithFit = callerFit
  ? items.map((d) => ({
      ...d,
      fitScorePercent: scoreToPercent(scoreDress(d, callerFit)),
      fitsCaller: passesFitsMeFilter(d, callerFit),
    }))
  : items.map((d) => ({ ...d, fitScorePercent: null, fitsCaller: null }));

return { items: itemsWithFit, total, page: input.page, limit: input.limit, callerHasMeasurements: !!callerFit };
```
