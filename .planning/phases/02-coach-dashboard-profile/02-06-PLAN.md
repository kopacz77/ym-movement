    - `approveProposal`: mutation. Input: `{ proposalId: z.string() }`.
      1. Fetch the ProposedTimeSlot by id
      2. Verify status is PENDING
      3. Transaction:
         a. Create a real RinkTimeSlot from the proposal data:
            ```typescript
            await tx.rinkTimeSlot.create({
              data: {
                rinkId: proposal.rinkId,
                coachId: proposal.coachId,
                startTime: proposal.startTime,
                endTime: proposal.endTime,
                maxStudents: proposal.maxStudents,
                isAvailable: true,
              },
            });
            ```
         b. Update ProposedTimeSlot: `status: APPROVED, reviewedAt: new Date(), reviewedById: ctx.session.user.id`
      4. Return the created time slot

    - `denyProposal`: mutation. Input: `{ proposalId: z.string(), notes: z.string().optional() }`.
      1. Update ProposedTimeSlot: `status: DENIED, adminNotes: input.notes ?? null, reviewedAt: new Date(), reviewedById: ctx.session.user.id`
      2. Return updated proposal
